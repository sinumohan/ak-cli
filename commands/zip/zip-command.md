### Zip a directory

`zip -r {{ZIP_FILE_NAME}} {{DIRECTORY_PATH}}`

- <b>ZIP_FILE_NAME</b>: Filename for new zipped file
- <b>DIRECTORY_PATH</b>: Directory relative/absolute path, which needs to be zipped

#### Example:

`zip -r ak-cli.zip ak-cli`
